# Roadmap

These are possible educational upgrades, not v1 features or commitments.

- Real IBM backend execution via a reviewed server-side Runtime adapter
- Noise models and ideal/noisy comparison
- Transpiler and coupling-map visualization
- Bloch sphere, statevector, and density matrix viewers
- Full BB84 mode with random bases, sifting, interception, and error rate
- Quantum algorithm teaching mode with guided explanations
- Accessible drag-and-drop, keyboard editing, and undo/redo
- Dynamic circuits, classical conditions, reset, controlled rotations, and symbols
- Import/export, reusable subcircuits, and process isolation for larger circuits

Any expansion should accept declarative data, validate it, and map it through explicit dispatch—never execute client-provided Python.
